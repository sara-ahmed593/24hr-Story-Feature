## Description

to build a "Story" feature similar to those found in popular social media platforms like Instagram and WhatsApp.
## Features

- There will be a list of stories at the top and a plus button.
- Clicking the plus button will allow user to upload an image which will be converted to base64 and stored in local storage.
- The image will be displayed in the list of stories.
- The image will be removed after 24 hours.
- User  optionally be able to swipe through the stories.
- story will close after 3s from opening it.
